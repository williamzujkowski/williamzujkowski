## How It Works

The animated SVG terminal is generated by [svg-terminal](https://github.com/williamzujkowski/svg-terminal). All content and configuration lives in `terminal.yml`.

**To update the terminal content:** Edit `terminal.yml` and push — the workflow runs automatically.

**To regenerate locally:**
```bash
# Install svg-terminal (one-time)
git clone https://github.com/williamzujkowski/svg-terminal.git /tmp/svg-terminal
cd /tmp/svg-terminal && npm ci && npm run build && npm link

# Generate
svg-terminal generate --config terminal.yml --output src/terminal.svg
```
